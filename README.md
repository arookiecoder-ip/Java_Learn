# Java_Learn

A repository for learning Java programming.

## About

This repository contains Java code examples, exercises, and projects created during the learning process of Java programming language.

## Contents

- Java fundamentals
- Object-oriented programming concepts
- Code examples and practice exercises

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- An IDE (IntelliJ IDEA, Eclipse, or VS Code recommended)

### Running the Code

1. Clone this repository:
   ```bash
   git clone https://github.com/arookiecoder-ip/Java_Learn.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Java_Learn
   ```

3. Compile and run Java files:
   ```bash
   javac YourFileName.java
   java YourFileName
   ```
